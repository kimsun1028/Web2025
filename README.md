# Fruit Game (Web2025)

간단한 숫자 조합 퍼즐 게임입니다. 사과(과일) 위의 숫자를 선택해 조합을 만들고 점수를 획득하세요. 스킨 선택, 업적 시스템, 최고 점수 저장, 사운드 효과 등을 지원합니다.

## 주요 기능
- 점수 시스템: 조합 성공 시 점수 획득, 최고 점수 저장(localStorage)
- 업적 시스템: 점수/특정 조합 달성 시 업적 해금, 토스트 알림
- 스킨 시스템: 다양한 과일 스킨 선택 및 적용
- 특수 과일: 시간 추가 과일, 골든 과일 등 특수 아이템
- 사운드: 배경음악 및 클릭/팝 효과음, 음소거 토글

## 폴더 구조
- 루트: `index.html`, `about.html`, `achievements.html`
- 게임: `Fruitgame/Fruitgame.html`, `Fruitgame/Fruitgame.js`, `Fruitgame/style.css`, `Fruitgame/image/`

## 실행 방법
1. 로컬에서 `index.html`을 브라우저로 열기
2. 시작 버튼으로 게임 화면(`Fruitgame/Fruitgame.html`) 이동
3. 과일을 선택해 조합을 만들고 점수를 올리세요
4. 업적 페이지에서 진행 상황을 확인할 수 있습니다

## 브라우저 권장사항
- 최신 크롬/엣지 계열 브라우저 권장
- 오디오 자동재생 정책에 따라 첫 상호작용 후 소리가 재생될 수 있습니다
